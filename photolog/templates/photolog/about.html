About

This site

This is a website designed for keeping up with family and friends while travelling in Europe and Asia, telling stories along with pictures.

Misc

This is an experimental website, hosted on pythonanywhere.com. If you have any request, bug report, or ideas of funky features, please do mail renon@disro*t.org with "o" for *.

This site was made from scratch, using the following (mainly open-source) :

  • Django, python web framework
  • jQuery, DOM animations
  • AJAX, asynchronous calls with jQuery
  • Vivus.js, svg animation
  • Git, version control system
  • OpenStreetMap, free editable world map
  • OpenLayers, JS library for OSM manipulation

The traveller

I am a french student, interested in computer stuff and oddities. On this matter, I deeply support open source based projects. From linux distros to basic softwares, it should be a major concern as many people fail to see what's going on beyond their screens, despite massively leaking personnal data.

Plenty of cool free stuff here : https://framasoft.org/

This is also the guarantee to have no "black box", eg. you can learn, understand and improve things you use every day, instead of enduring them. By the way, source code of this site is availabe on this repo :
